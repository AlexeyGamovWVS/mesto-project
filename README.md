# Проект: Место

### Обзор

Проект супер-упрощённого инстаграм для работы с JS. Позволяет загружать фотографии, лайкать и обновлять информацию о профиле. [Смотреть проект] (https://alexeygamovwvs.github.io/mesto-project/)

### Технологии:

* БЭМ Nested
* Адаптивная вёрстка
* JavaScript
* Flexbox / Grid Layout
